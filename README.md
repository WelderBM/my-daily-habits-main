## 🇧🇷 Português
 
### Sobre
 
**My Daily Habits** é uma aplicação de rastreamento de hábitos diários desenvolvida com React. O usuário pode adicionar e remover hábitos, acompanhar o progresso diário e semanal, e manter seus dados persistidos entre sessões.
 
Este projeto foi construído ao longo das atividades práticas do curso de React no ITEAM/CCTI, aplicando conceitos fundamentais do ecossistema React moderno em um projeto funcional.
 
### Funcionalidades
 
- Adicionar e remover hábitos personalizados
- Acompanhamento de progresso diário e semanal
- Persistência de dados (os hábitos são mantidos entre sessões)
- Interface responsiva para desktop e mobile
- Navegação entre páginas com React Router
 
### Stack
 
| Tecnologia | Versão | Função |
|---|---|---|
| React | 19 | Biblioteca de UI |
| React Router DOM | 7 | Navegação SPA |
| Vite | 8 | Build tool e dev server |
| JavaScript (ES6+) | — | Linguagem principal |
| CSS3 | — | Estilização |
| ESLint | 9 | Linting e qualidade de código |
 
### Como rodar
 
```bash
# Clone o repositório
git clone https://github.com/WelderBM/my-daily-habits-main.git
 
# Entre na pasta do projeto
cd my-daily-habits-main
 
# Instale as dependências
npm install
 
# Inicie o servidor de desenvolvimento
npm run dev
```
 
O app estará disponível em `http://localhost:5173`
 
### Scripts disponíveis
 
| Comando | Descrição |
|---|---|
| `npm run dev` | Inicia o servidor de desenvolvimento |
| `npm run build` | Gera o build de produção |
| `npm run preview` | Pré-visualiza o build de produção |
| `npm run lint` | Executa o ESLint |
 
### Aprendizados
 
Este projeto consolidou conceitos como:
- Componentização e reutilização de componentes React
- Gerenciamento de estado com hooks (`useState`, `useEffect`)
- Roteamento SPA com React Router DOM
- Persistência de dados no lado do cliente
- Estilização responsiva com CSS puro
- Configuração de projeto com Vite e ESLint

## 🇺🇸 English
 
### About
 
**My Daily Habits** is a daily habit tracking application built with React. Users can add and remove habits, track daily and weekly progress, and keep their data persisted between sessions.
 
This project was developed throughout practical activities in the React course at ITEAM/CCTI, applying fundamental concepts of the modern React ecosystem in a functional project.
 
### Features
 
- Add and remove custom habits
- Daily and weekly progress tracking
- Data persistence (habits are kept between sessions)
- Responsive interface for desktop and mobile
- Page navigation with React Router
 
### Stack
 
| Technology | Version | Purpose |
|---|---|---|
| React | 19 | UI library |
| React Router DOM | 7 | SPA navigation |
| Vite | 8 | Build tool & dev server |

```bash
# Clone the repository
git clone https://github.com/WelderBM/my-daily-habits-main.git
```
